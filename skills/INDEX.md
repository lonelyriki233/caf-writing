## 完整工程流（Strict SOP）

```
premise/compass → story engine(主题/行动)
 → volume/act/sequence → character engine
 → chapter spec → scene cards
 → dialogue maps → draft
 → anti-AI/prose audit
 → objective quality gate → revision report
```

每个阶段的模板和诊断均在 `skills/novel_kb_ref/` 下。
