# Prose and Revision Pattern Family

Sources: Le Guin, Prose, Klinkenborg, Browne/King, Stephen King.

## Core pattern
Story function first → viewpoint pressure → concrete detail → sentence rhythm → dialogue mechanics → compression → final style pass.

## Rule
Beautiful prose is not allowed to carry an empty scene. Prose polish happens after scene/story/dialogue gates pass.

## Mandatory gates
- detail reveals pressure or perception
- paragraph rhythm follows emotional/action pressure
- dialogue beats reveal status/evasion/tension
- exposition appears only at point of need
- revision pass declares layer: structure / scene / dialogue / interiority / line

## Failure labels
- smooth-but-empty
- overexplained emotion
- generic description
- rhythm uniform
- tag/adverb explaining
