# Plot Traction Pattern Family

Sources: McKee Story, Truby Anatomy, Bell Plot & Structure, Brooks Story Engineering, Weiland Structuring.

## Core pattern
Disturbance → commitment doorway → reaction under wrong strategy → midpoint context shift → attack/agency → low point/crisis choice → climax → aftermath.

## Objective quality standard
A plot beat is weak if it only adds events. It must change pressure, information, value charge, agency, relationship, or cost.

## Mandatory gates
- external want exists
- failure cost is concrete
- opponent/counterforce attacks want and weakness
- midpoint changes context or agency
- climax forces choice, not only victory

## Fail labels
- event-list plot
- soft midpoint
- no-return missing
- antagonist disappears
- ending resolves facts but not pressure
