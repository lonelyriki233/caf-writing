# Character Arc Pattern Family

Sources: Weiland Character Arcs, Cron Wired for Story, Kress, Card, Truby.

## Core pattern
Wound/internal issue → lie/wrong strategy → external want → pressure exposes contradiction → repeated costly choices → crisis truth/refusal → changed action/unchanged tragic commitment.

## Mandatory gates
- want/need split
- lie or internal issue visible in action
- opponent or relationship pressures the gap
- arc beat aligns with plot beat
- final action proves truth/refusal

## Failure labels
- label-character
- convenient change
- backstory inert
- emotional jump
- relationship has no pressure function
