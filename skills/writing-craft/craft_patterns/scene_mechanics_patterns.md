# Scene Mechanics Pattern Family

Sources: Swain, Bickham, McKee, Weiland, Bell.

## Core pattern
Stimulus → response → goal → conflict → tactic shift → result/turn → sequel reaction → decision → next scene goal.

## Use when
- scene is correct but dull
- characters talk without pressure
- chapter ends without hook
- plot feels episodic

## Mandatory gates
1. Target: what does POV want now?
2. Resistance: what blocks it actively?
3. Tactic: what does POV do?
4. Turn: what changes?
5. Sequel: what emotion/thought/decision launches next?

## Anti-copy rule
Do not copy any scene surface from source works. Extract only causal mechanics.
