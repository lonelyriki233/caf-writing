# Objective Quality Gate Upgrade

This upgrades previous “usable” gates toward objective craft judgment.

## Strong chapter definition
A chapter is strong only if:
1. It opens with imbalance/question/pressure.
2. The protagonist or focal character pursues something.
3. Resistance forces tactic change.
4. At least one scene result changes the situation.
5. Dialogue, if present, has verbal action/subtext.
6. Exposition arrives because current action needs it.
7. The ending produces a new question, decision, danger, cost, or emotional residue.
8. Prose supports pressure through concrete detail and rhythm.

## Hard fail
Any of these blocks drafting or marks revision failure:
- no protagonist/focal pressure
- no scene result
- no cost
- dialogue as Q&A exposition
- worldbuilding before trouble
- ending with summary only

## Objective scoring
Use `quality_gates/QUALITY_RUBRICS.md`, but add:
- scene mechanics score
- plot traction score
- character arc pressure score
- prose function score

Target: no core dimension below 2; at least one dimension at 3 per important chapter.
