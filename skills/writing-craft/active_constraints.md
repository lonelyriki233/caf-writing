# Active Constraints

These are durable constraints for novel_lab work.

1. Do not draft prose first. Prose is allowed only after story/scene/dialogue gates.
2. Do not use style polish to hide weak story engine.
3. Dialogue is verbal action, not Q&A exposition.
4. Every key scene must have target, obstacle, result, and changed situation.
5. Exposition must be timed to current action need.
6. NPCs that affect a scene need voice cards: job/current pressure/fear/attitude/action verb.
7. If user criticizes a failure twice, convert it into a named diagnostic and add it here.
