# Dialogue Subtext Checklist

- [ ] Each important speaker has a want.
- [ ] Each important line has an action verb.
- [ ] Said / unsaid / unsayable are identified for emotional, sacred, political, or secret topics.
- [ ] Dialogue changes relationship, information, pressure, or decision.
- [ ] Exposition appears through conflict/procedure/mistake/taboo/need, not lecture.
- [ ] NPC has voice card if more than functional.
