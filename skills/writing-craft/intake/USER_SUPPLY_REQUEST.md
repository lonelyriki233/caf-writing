# User Supply Request / 需要你补的材料

为了从“可用”推进到“用得好”，最缺的是审美靶心和案例样本。

## Highest-value materials
1. 你最喜欢的小说/轻小说/gal 文本片段或章节，3-10 个。
   - 最好说明：喜欢哪里？开头、对白、情绪、悬念、人物、节奏？
2. 你最讨厌的失败样本，3-10 个。
   - 可以是我写烂的，也可以是别的文本。
   - 说明：哪里无聊、假、AI、尬、拖、散？
3. 你当前最想写的项目方向。
   - 类型、主角、核心关系、第一卷目标、你想要读者获得的感觉。
4. 对标作品。
   - 不用整本都给，给章节名/片段/路径即可。
