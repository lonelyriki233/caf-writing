# Gates

- [ ] 能说出至少3个来自生活观察的可见细节，且它们会改变人物处境或选择。 (生活材料优先)
- [ ] 删掉主题说明后，读者仍能从人物选择和后果中读出核心问题。 (人物先于主题)
- [ ] 小场面能折射更大处境，并且主角必须在其中付出代价。 (以小见大)
- [ ] 用职业/身份标签替换人物名后，场景仍不应成立；若成立，人物还不够具体。 (典型来自个性)
- [ ] 每个关键判断旁边都有可见行动或可检验后果支撑。 (行动塑造人物)
- [ ] 心理句删去后，动作链仍能传达情绪方向；保留的心理句会推动选择。 (心理附着外部反应)
- [ ] 每个环境细节都能回答：它让人物更难/更暴露/更被迫选择了吗？ (环境服务人物情绪)
- [ ] 一页内抽象评价词明显减少，至少新增一个能被镜头拍到的细节。 (语言去概念化)
