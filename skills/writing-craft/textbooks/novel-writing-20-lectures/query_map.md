# Query Map

- If `构思只有概念、口号、题材标签时`, query `生活材料优先`; apply gate: 能说出至少3个来自生活观察的可见细节，且它们会改变人物处境或选择。
- If `主题句已经明确但人物还只是功能标签时`, query `人物先于主题`; apply gate: 删掉主题说明后，读者仍能从人物选择和后果中读出核心问题。
- If `题材很大但情节散、人物远时`, query `以小见大`; apply gate: 小场面能折射更大处境，并且主角必须在其中付出代价。
- If `人物被写成阶层、职业、性格标签时`, query `典型来自个性`; apply gate: 用职业/身份标签替换人物名后，场景仍不应成立；若成立，人物还不够具体。
- If `段落主要在评价人物“善良/复杂/痛苦”时`, query `行动塑造人物`; apply gate: 每个关键判断旁边都有可见行动或可检验后果支撑。
- If `心理描写悬空抒情、脱离场景时`, query `心理附着外部反应`; apply gate: 心理句删去后，动作链仍能传达情绪方向；保留的心理句会推动选择。
- If `环境描写只是风景或资料展示时`, query `环境服务人物情绪`; apply gate: 每个环境细节都能回答：它让人物更难/更暴露/更被迫选择了吗？
- If `文本出现套话、抽象判断、空泛抒情时`, query `语言去概念化`; apply gate: 一页内抽象评价词明显减少，至少新增一个能被镜头拍到的细节。
