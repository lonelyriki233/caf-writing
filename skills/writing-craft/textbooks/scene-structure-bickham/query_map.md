# Query Map

- If `scene opens without directional want`, query `scene-unit-goal`; apply gate: scene_card gate: target required.
- If `obstacle is solved too easily`, query `conflict-escalation`; apply gate: scene_card gate: at least one tactic shift.
- If `scene ends where it began`, query `disaster-or-turn`; apply gate: scene_card gate: result must change situation.
- If `reaction is skipped`, query `sequel-emotion-thought`; apply gate: chapter_spec gate: major disaster needs processing beat.
- If `next scene feels arbitrary`, query `decision-action-bridge`; apply gate: volume_structure gate: scene chain must be causal.
- If `dialogue/actions feel scrambled`, query `stimulus-response-order`; apply gate: draft gate: fix reversed causality.
- If `scene viewpoint drifts`, query `viewpoint-consistency`; apply gate: draft gate: POV shift must have function.
- If `chapter ends with summary`, query `chapter-ending-hook`; apply gate: chapter_spec gate: closing turn required.
