# Query Map

- If `events happen but do not matter`, query `story-as-meaning`; apply gate: story_engine gate: why it matters required.
- If `setup is generic`, query `specificity`; apply gate: draft gate: detail must carry story function.
- If `plot external only`, query `internal-issue`; apply gate: character_engine gate: gap/internal issue.
- If `episodes disconnected`, query `cause-effect`; apply gate: volume_structure gate: causal chain.
- If `stakes big but impersonal`, query `stakes-personal`; apply gate: story_engine gate: failure cost personal.
- If `chapter lacks curiosity`, query `reader-question`; apply gate: chapter_spec gate: new question required.
- If `flaw told not shown`, query `misbelief-in-action`; apply gate: scene_card gate: tactic reveals belief.
- If `emotion label feels cheap`, query `emotion-from-context`; apply gate: prose_audit gate: emotion embodied.
