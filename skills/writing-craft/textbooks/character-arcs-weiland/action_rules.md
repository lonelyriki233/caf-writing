# Action Rules

## lie-character-believes
Trigger: character flaw vague

Rule: Define the false belief/lie shaping choices.

SOP gate: character_engine gate: wrong strategy/lie required.

## truth-opposes-lie
Trigger: arc has no destination

Rule: Define the truth the plot pressures the character to accept/refuse.

SOP gate: theme_action gate: theme as tested truth.

## want-vs-need
Trigger: goal and growth identical

Rule: Separate external want from internal need.

SOP gate: story_engine gate: want/need split.

## wound-pressure
Trigger: backstory inert

Rule: Use wound only if it drives present false strategy.

SOP gate: character_engine gate: wound maps to action.

## positive-change
Trigger: hero changes too suddenly

Rule: Escalate lie pressure until choice proves new truth.

SOP gate: volume_structure gate: arc beats align with plot beats.

## flat-arc
Trigger: mentor/protagonist static by accident

Rule: Flat arc character holds truth and changes world; test conviction under pressure.

SOP gate: character_engine gate: static character still pressured.

## negative-arc
Trigger: fall feels arbitrary

Rule: Show repeated refusal of truth and increasing commitment to lie.

SOP gate: revision gate: choices track decline.

## relationship-mirror
Trigger: supporting characters decorative

Rule: Use allies/opponents as mirrors of lie/truth variations.

SOP gate: character_engine gate: relationship function.
