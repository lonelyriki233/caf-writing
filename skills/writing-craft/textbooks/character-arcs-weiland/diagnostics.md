# Diagnostics

## Symptom: character flaw vague
Repair: Define the false belief/lie shaping choices.
Gate: character_engine gate: wrong strategy/lie required.

## Symptom: arc has no destination
Repair: Define the truth the plot pressures the character to accept/refuse.
Gate: theme_action gate: theme as tested truth.

## Symptom: goal and growth identical
Repair: Separate external want from internal need.
Gate: story_engine gate: want/need split.

## Symptom: backstory inert
Repair: Use wound only if it drives present false strategy.
Gate: character_engine gate: wound maps to action.

## Symptom: hero changes too suddenly
Repair: Escalate lie pressure until choice proves new truth.
Gate: volume_structure gate: arc beats align with plot beats.

## Symptom: mentor/protagonist static by accident
Repair: Flat arc character holds truth and changes world; test conviction under pressure.
Gate: character_engine gate: static character still pressured.

## Symptom: fall feels arbitrary
Repair: Show repeated refusal of truth and increasing commitment to lie.
Gate: revision gate: choices track decline.

## Symptom: supporting characters decorative
Repair: Use allies/opponents as mirrors of lie/truth variations.
Gate: character_engine gate: relationship function.
