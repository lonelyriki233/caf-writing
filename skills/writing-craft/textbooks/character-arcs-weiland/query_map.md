# Query Map

- If `character flaw vague`, query `lie-character-believes`; apply gate: character_engine gate: wrong strategy/lie required.
- If `arc has no destination`, query `truth-opposes-lie`; apply gate: theme_action gate: theme as tested truth.
- If `goal and growth identical`, query `want-vs-need`; apply gate: story_engine gate: want/need split.
- If `backstory inert`, query `wound-pressure`; apply gate: character_engine gate: wound maps to action.
- If `hero changes too suddenly`, query `positive-change`; apply gate: volume_structure gate: arc beats align with plot beats.
- If `mentor/protagonist static by accident`, query `flat-arc`; apply gate: character_engine gate: static character still pressured.
- If `fall feels arbitrary`, query `negative-arc`; apply gate: revision gate: choices track decline.
- If `supporting characters decorative`, query `relationship-mirror`; apply gate: character_engine gate: relationship function.
