# 救猫咪：小说创作指南 — action rules

Source: `/mnt/f/创作/奇迹/novel_lab/books/写作教科书/extracted/save-the-cat-novel.txt`

## Use this module when
- 长篇结构不清，只有事件串联。
- 主角缺少 Want/Need，结尾没有转变重量。
- 类型承诺、节拍压力或推销文案不清楚。

## Rules
1. 主角先有问题、Want、Need：外部追逐与内在课程必须同时存在。
2. 情节、结构、角色转变一起设计；事件顺序要逼迫改变。
3. 用15动点检查路线，但不要机械表演模板。
4. 催化剂必须撞出原轨道。
5. 进入第二幕通常是主角带着缺陷选择的错误方案。
6. 故事B承载内在旅程，反照或推动 Need。
7. 中点必须改变压力等级，形成虚假胜利或虚假失败。
8. 坏人逼近阶段同步升级外部风险和内部崩塌。
9. 一切尽失要击碎旧策略。
10. 结局让主角用新的自我做最终选择。
11. 按十种故事类型补足类型核心元素。
12. 用一句话简介检验故事是否清楚可卖。

## Gates
- 主角 Want 与 Need 是否能分别说清？
- 中点后故事规则是否明显改变？
- 最终行动是否同时解决外部目标并证明内在转变？
