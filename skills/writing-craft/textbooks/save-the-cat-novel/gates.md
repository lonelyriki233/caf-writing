# Gates

- [ ] 三项都能写成一句话，且 Want 与 Need 存在张力。 (值得写的主角三件套)
- [ ] 每个大事件都能说明它如何逼近或阻碍主角转变。 (情节结构角色转变三位一体)
- [ ] 每个动点不是标签，而能对应具体场景/段落功能。 (15动点路线图)
- [ ] 催化剂后主角若不行动，损失会扩大或局面不可逆。 (催化剂撞出原轨道)
- [ ] 第二幕世界与第一幕有明显差异，且主角策略带着缺陷。 (进入第二幕是错误方案)
- [ ] 故事B删除后，主角内在转变会变薄或失去触发。 (故事B承载内在旅程)
- [ ] 中点后故事不能按原速度和原规则继续。 (中点改变压力级别)
- [ ] 每次外部升级都暴露主角缺点或错误信念的新裂缝。 (坏人逼近双线升级)
- [ ] 主角不能靠第一幕/第二幕的旧策略合理解决低谷。 (一切尽失击碎旧我)
- [ ] 最终行动同时解决外部目标并证明 Need 已被领会。 (进入第三幕靠新选择)
- [ ] 读者能清楚感到该类型的前提承诺，并在结尾获得对应满足。 (十种故事类型元素)
- [ ] 一句话包含主角缺陷、第二幕动作、中点转折、失去一切风险和主题学习。 (一句话简介模板)
