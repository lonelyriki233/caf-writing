# Query Map

- If `opening explains background`, query `dont-explain-too-soon`; apply gate: draft gate: no encyclopedia opening.
- If `plot resolves by convenience`, query `dont-make-things-easy`; apply gate: story_engine gate: obstacle must actively resist.
- If `scene pauses for lecture`, query `dont-stop-story`; apply gate: prose_audit gate: exposition must be dramatized.
- If `dialogue sounds conversational but pointless`, query `dont-use-empty-dialogue`; apply gate: dialogue_map gate: action verb per speaker.
- If `character choice feels arbitrary`, query `dont-forget-motivation`; apply gate: character_engine gate: choice maps to want/need/wrong strategy.
- If `hero avoids real loss`, query `dont-overprotect-protagonist`; apply gate: story_engine gate: failure cost required.
- If `draft treated as final`, query `dont-neglect-revision`; apply gate: revision gate: name changed layer.
- If `nothing changes`, query `dont-write-static-scenes`; apply gate: scene_card gate: changed situation required.
