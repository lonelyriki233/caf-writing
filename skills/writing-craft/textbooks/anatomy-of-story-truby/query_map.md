# Query Map

- If `story premise is just setup`, query `designing-principle`; apply gate: compass gate: project north star required.
- If `protagonist has goal but no flaw/need`, query `weakness-need`; apply gate: character_engine gate: gap/need required.
- If `plot lacks spine`, query `desire-line`; apply gate: story_engine gate: external want required.
- If `villain is obstacle only`, query `opponent-as-necessary`; apply gate: character_engine gate: opposition maps to protagonist gap.
- If `supporting cast random`, query `character-web`; apply gate: character_engine gate: relationship function required.
- If `theme preached`, query `moral-argument`; apply gate: theme_action gate: at least three tested situations.
- If `middle wanders`, query `plan-and-counterplan`; apply gate: volume_structure gate: sequence goals/counterpressure required.
- If `ending changes facts only`, query `self-revelation`; apply gate: revision gate: final outcome maps to need.
