# Query Map

- If `story lacks central pull`, query `lock-lead`; apply gate: story_engine gate: protagonist agency required.
- If `chapters feel aimless`, query `lock-objective`; apply gate: story_engine gate: external want required.
- If `opposition too soft`, query `lock-confrontation`; apply gate: volume_structure gate: act II route is confrontation.
- If `ending fizzles`, query `lock-knockout`; apply gate: volume_structure gate: finale choice + consequence.
- If `opening starts normal too long`, query `disturbance`; apply gate: chapter_spec gate: opening imbalance required.
- If `act breaks are weak`, query `doorways-no-return`; apply gate: volume_structure gate: plot point changes commitment.
- If `stakes are vague`, query `death-stakes`; apply gate: story_engine gate: failure cost concrete.
- If `character could walk away`, query `adhesive`; apply gate: story_engine gate: why-now/why-stay required.
