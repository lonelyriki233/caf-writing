# Query Map

- If `scene feels floaty or reactions seem unmotivated`, query `mru-stimulus-response`; apply gate: scene_card gate: every important reaction must have visible stimulus and consequence.
- If `scene only describes events`, query `scene-goal-conflict-disaster`; apply gate: scene_card gate: no scene passes without goal-conflict-result.
- If `after a disaster the story jumps mechanically`, query `sequel-reaction-dilemma-decision`; apply gate: chapter_spec gate: sequel must produce next action.
- If `characters act because author needs plot`, query `motivation-reaction-continuity`; apply gate: draft gate: reject uncaused actions.
- If `chapter has no forward pull`, query `desire-danger-tension`; apply gate: story_engine gate: want and cost cannot be abstract.
- If `POV lacks immediacy`, query `viewpoint-pressure`; apply gate: draft gate: viewpoint detail must serve pressure.
- If `obstacle is passive`, query `conflict-as-resistance`; apply gate: scene_card gate: obstacle must force tactic change.
- If `chapters feel episodic`, query `decision-creates-next-scene`; apply gate: chapter_spec gate: closing turn must generate next question/action.
