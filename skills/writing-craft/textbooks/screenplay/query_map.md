# Query Map

- If `准备章、卷、场景或任务线时，只能说出题材/设定/世界观，不能说出谁在做什么。`, query `Subject = character + action`; apply gate: 一句话能否回答：谁在做什么？该行动是否会改变目标、关系、风险或信息？若只能回答‘展示/介绍/说明’，不通过。
- If `有灵感但进入大纲时发散成多条设定、背景史或概念主题。`, query `Subject sentence before expansion`; apply gate: 大纲第一行是否包含主角、行动、处境、变化方向？所有新增场景是否能回扣这句话？
- If `章节像流水账，开端、中段、结尾只是篇幅位置，不承担不同压力功能。`, query `Three acts as pressure phases`; apply gate: 每个阶段是否有不同问题：建立什么、对抗什么、解决什么？阶段转换后主角能否回到原状？能回到则不通过。
- If `开头先铺历史、制度、世界观，迟迟不建立主角处境和关系。`, query `Act I / Setup context`; apply gate: 开头是否让读者知道主角是谁、现在处在什么麻烦里、TA与谁发生关系张力？若主要内容是百科说明，不通过。
- If `第一章没有可追踪目标、现实阻碍或下一章问题。`, query `First ten pages / first chapter hook`; apply gate: 前3000字能否标注主角、目标、阻碍、悬而未决问题？读者是否有继续读的具体理由？
- If `中段只是任务堆叠，结尾只是停止事件而非解决行动线。`, query `Act II confrontation and Act III solution`; apply gate: 中段每个序列是否让主角更难完成需要？结尾是否回答了核心行动问题：成功/失败/代价/选择？
- If `主角只是旁观设定、接任务、听解释，缺少非完成不可的需要。`, query `Dramatic need drives the story`; apply gate: 能否写出‘主角必须____，否则____’？若没有失败代价或时限压力，不通过。
- If `章节结尾有事件但没有改变后续行动方向。`, query `Plot point turns action into a new direction`; apply gate: plot point 之后主角还能按原计划安全返回吗？若能，则只是事件不是情节点。
- If `转折发生在远处、反派或设定层，主角行动没有被迫改变。`, query `Plot point belongs to main character function`; apply gate: 能否说明此转折如何改变主角下一步行动？若只是读者知道新信息而主角无变化，不通过。
- If `长篇连续多章无转向，或每章都用假悬念但主线不前进。`, query `Multiple plot points as anchor and cadence`; apply gate: 最近3-5章是否出现不可回撤的状态变化？如果只有战斗/解释/插科打诨而行动线不变，不通过。
- If `某场戏只是气氛、设定讲解、闲聊或作者喜欢的桥段。`, query `Scene purpose: move story or reveal character`; apply gate: 场景结束时故事是否前进？角色是否暴露了必要的新信息/选择/矛盾？两者皆无则不通过。
- If `场景边写边走，没有明确目标、阻碍和结果。`, query `Scene as target-obstacle-result unit`; apply gate: 能否用四行列出目标、阻碍、结果、下一问题？若结果是‘他们聊完/了解设定’，不通过。
- If `一场戏塞入多段制度、背景、人物史、暗线解释。`, query `One necessary information per scene`; apply gate: 本场的唯一必需信息是什么？删掉其余解释后读者是否仍能继续跟行动？若信息超过一个且拖慢行动，不通过。
- If `章节由散点事件组成，缺少统一小目标、小高潮或小转折。`, query `Sequence as block of dramatic action`; apply gate: 该 sequence 能否用一个短语命名？是否有开始、中段升级、结束转向？其中每场是否服务同一小目标？
- If `先列场景再试图找意义，导致行动段落松散。`, query `Sequence context before content`; apply gate: 是否先写出 Sequence A goal/context？删掉偏离 context 的场景后，段落是否更紧？
- If `动作场面压倒人物，或人物心理独白脱离行动。`, query `Action and character must balance`; apply gate: 动作是否让人物发生选择/暴露/代价？人物变化是否通过行动表现？若动作可替换任意角色，不通过。
- If `写法依赖作者旁白解释主题、制度或情绪，缺少可观察行为。`, query `Visual/action presentation over abstract explanation`; apply gate: 该信息是否能被读者从行为/对话压力/物件变化中推断？若只能靠作者说明，不通过。
- If `直接写正文导致不知道写什么，修稿只改句子不改结构。`, query `Preparation then execution`; apply gate: 正文前是否已有章节规格？修稿报告是否先处理结构问题而非句子润色？
