# Gates

- [ ] 一句话能否回答：谁在做什么？该行动是否会改变目标、关系、风险或信息？若只能回答‘展示/介绍/说明’，不通过。 (Subject = character + action)
- [ ] 大纲第一行是否包含主角、行动、处境、变化方向？所有新增场景是否能回扣这句话？ (Subject sentence before expansion)
- [ ] 每个阶段是否有不同问题：建立什么、对抗什么、解决什么？阶段转换后主角能否回到原状？能回到则不通过。 (Three acts as pressure phases)
- [ ] 开头是否让读者知道主角是谁、现在处在什么麻烦里、TA与谁发生关系张力？若主要内容是百科说明，不通过。 (Act I / Setup context)
- [ ] 前3000字能否标注主角、目标、阻碍、悬而未决问题？读者是否有继续读的具体理由？ (First ten pages / first chapter hook)
- [ ] 中段每个序列是否让主角更难完成需要？结尾是否回答了核心行动问题：成功/失败/代价/选择？ (Act II confrontation and Act III solution)
- [ ] 能否写出‘主角必须____，否则____’？若没有失败代价或时限压力，不通过。 (Dramatic need drives the story)
- [ ] plot point 之后主角还能按原计划安全返回吗？若能，则只是事件不是情节点。 (Plot point turns action into a new direction)
- [ ] 能否说明此转折如何改变主角下一步行动？若只是读者知道新信息而主角无变化，不通过。 (Plot point belongs to main character function)
- [ ] 最近3-5章是否出现不可回撤的状态变化？如果只有战斗/解释/插科打诨而行动线不变，不通过。 (Multiple plot points as anchor and cadence)
- [ ] 场景结束时故事是否前进？角色是否暴露了必要的新信息/选择/矛盾？两者皆无则不通过。 (Scene purpose: move story or reveal character)
- [ ] 能否用四行列出目标、阻碍、结果、下一问题？若结果是‘他们聊完/了解设定’，不通过。 (Scene as target-obstacle-result unit)
- [ ] 本场的唯一必需信息是什么？删掉其余解释后读者是否仍能继续跟行动？若信息超过一个且拖慢行动，不通过。 (One necessary information per scene)
- [ ] 该 sequence 能否用一个短语命名？是否有开始、中段升级、结束转向？其中每场是否服务同一小目标？ (Sequence as block of dramatic action)
- [ ] 是否先写出 Sequence A goal/context？删掉偏离 context 的场景后，段落是否更紧？ (Sequence context before content)
- [ ] 动作是否让人物发生选择/暴露/代价？人物变化是否通过行动表现？若动作可替换任意角色，不通过。 (Action and character must balance)
- [ ] 该信息是否能被读者从行为/对话压力/物件变化中推断？若只能靠作者说明，不通过。 (Visual/action presentation over abstract explanation)
- [ ] 正文前是否已有章节规格？修稿报告是否先处理结构问题而非句子润色？ (Preparation then execution)
