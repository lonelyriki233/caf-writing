# Gates

- [ ] 第一页或第一场能指出具体麻烦及其对人物的风险。 (麻烦缠身的人物)
- [ ] 主角失败会失去生命、安全、事业/身份，或完整自我中的一项。 (死亡赌注三分法)
- [ ] 四项各能用一句话说明；缺一项则先补基础再扩写场景。 (LOCK基础)
- [ ] 开头场面中有行动中的人物和扰乱其日常的事件。 (以乱子开头)
- [ ] 读者能说明主角为什么不能回到原状。 (无回关口)
- [ ] 场景结束后局势变化；删掉该场会破坏因果链。 (场景目标障碍结果)
- [ ] 每个反应场景都连接总体目标与下一步行动。 (行动与反应节拍)
- [ ] 至少两处能看到支线改变主角对主线的行动或代价。 (子情节入侵主线)
- [ ] 每段关键对话后，关系、信息或权力位置发生偏转。 (对话制造冲突)
- [ ] 每章结尾至少留下一个与赌注相关的未解决问题。 (宏观悬念)
- [ ] 结尾迫使读者关心后果，但不破坏全书最终满足感。 (悬崖挂壁变化)
- [ ] 障碍虽小，却会让人物目标更难、时间更紧或秘密更危险。 (微观障碍与张力)
