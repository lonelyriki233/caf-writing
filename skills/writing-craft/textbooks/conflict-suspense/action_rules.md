# 冲突与悬念 — action rules

Source: `/mnt/f/创作/奇迹/novel_lab/books/写作教科书/extracted/conflict-suspense.txt`

## Use this module when
- 故事缺少推进力，开头平，章节结尾不勾人。
- 场景只是说明、聊天、铺背景。
- 中段疲软，支线散，悬念只靠藏信息。

## Rules
1. 从麻烦开始：人物必须遇到对命运重要的麻烦。
2. 明确死亡赌注：肉体、职场/使命或心理死亡至少占一项。
3. 先搭 LOCK：主角、目标、对抗、精彩结尾缺一不可。
4. 开头先乱子后解释：行动优先，展示延后。
5. 设置无回关口：主角必须进入主要对抗，不能轻易退回原状。
6. 每场戏写目标、障碍、结果；结果最好带来挫败或新问题。
7. 对话要有不同盘算、潜台词、偏转和关系变化。
8. 悬念是未闭合问题：让读者持续追问接下来会怎样。
9. 章节结尾变化使用悬崖挂壁，不要只重复同一种危机。
10. 在场景中插入微观障碍和延迟，让等待产生张力。

## Gates
- 主角为什么不能退出？
- 每场戏删掉后，因果链是否会断？
- 每章结尾是否留下与赌注相关的未解决问题？
