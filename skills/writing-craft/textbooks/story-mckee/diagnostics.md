# Diagnostics

## Symptom: scene event is factual only
Repair: Track value change from positive to negative or reverse; story movement is value movement.
Gate: scene_card gate: identify value at start/end.

## Symptom: beat list is vague
Repair: Break scenes into action/reaction beats where tactics change under resistance.
Gate: scene_card gate: beat turns required in key scenes.

## Symptom: plot point is just event
Repair: A turning point must reverse expectation, reveal gap, or force new action.
Gate: volume_structure gate: plot point changes strategy/pressure.

## Symptom: middle repeats pressure
Repair: Escalate complications by making each attempt harder, more personal, or more costly.
Gate: volume_structure gate: no flat repetition.

## Symptom: events feel predictable
Repair: Design gaps between what character expects and what reality delivers.
Gate: scene_card gate: result should surprise character logically.

## Symptom: climax is external only
Repair: Force a crisis choice between irreconcilable goods/bads before climax action.
Gate: volume_structure gate: final choice cost required.

## Symptom: theme is slogan
Repair: Express theme as value plus cause proven by action outcome.
Gate: theme_action gate: theme tested by consequences.

## Symptom: backstory dumps
Repair: Use exposition only when it arms current conflict or changes interpretation.
Gate: draft gate: info must have now-function.
