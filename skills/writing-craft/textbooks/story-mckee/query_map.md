# Query Map

- If `scene event is factual only`, query `value-charge`; apply gate: scene_card gate: identify value at start/end.
- If `beat list is vague`, query `beat-as-action-reaction`; apply gate: scene_card gate: beat turns required in key scenes.
- If `plot point is just event`, query `turning-point`; apply gate: volume_structure gate: plot point changes strategy/pressure.
- If `middle repeats pressure`, query `progressive-complication`; apply gate: volume_structure gate: no flat repetition.
- If `events feel predictable`, query `gap-between-expectation-result`; apply gate: scene_card gate: result should surprise character logically.
- If `climax is external only`, query `crisis-choice`; apply gate: volume_structure gate: final choice cost required.
- If `theme is slogan`, query `controlling-idea`; apply gate: theme_action gate: theme tested by consequences.
- If `backstory dumps`, query `exposition-as-ammunition`; apply gate: draft gate: info must have now-function.
