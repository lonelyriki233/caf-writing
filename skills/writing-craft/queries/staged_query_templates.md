# Staged Query Templates

Use these before drafting or revising.

## Story engine query
- What does the protagonist want externally?
- What do they need internally or strategically?
- What pressure makes action necessary now?
- What failure cost makes the chapter/story worth reading?
- Which old strategy will fail?

## Structure query
- Which act/sequence is this?
- What is the local goal?
- What turn changes the situation?
- What question pulls the reader forward?

## Scene query
- Scene target:
- Obstacle:
- Tactic/action:
- Result:
- New question:
- If deleted, what changes?

## Dialogue query
- Speaker A wants:
- Speaker B wants:
- Action verbs:
- Said:
- Unsaid:
- Unsayable:
- Exposition allowed now:
- Exposition delayed:

## Prose audit query
- Is the paragraph carrying story action or only atmosphere?
- Is emotion embodied before labeled?
- Does the sentence rhythm match pressure?
- Does the scene end with residue/consequence/question?
