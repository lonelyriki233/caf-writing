# Dialogue Failure Diagnostics

## Symptom: 对话像机器人/问什么答什么
Likely causes:
- no action verb
- no subtext
- all speakers share author voice
Repair:
- assign each line a verb: probe, hide, refuse, soothe, pressure, deflect, bargain
- write said / unsaid / unsayable before prose

## Symptom: 对白在讲设定
Likely causes:
- exposition not tied to current need
- no conflict around information
Repair:
- reveal through form, procedure, correction, taboo, mistake, or negotiation
- give minimum needed now, delay the rest

## Symptom: NPC 没有人味
Likely causes:
- no job/current pressure/attitude
- only exists to deliver info
Repair:
- add NPC voice card: job, age-position, current pressure, fear, attitude, speech habit, action verb

## Symptom: 神圣/悼念对象被轻浮处理
Likely causes:
- missing unsayable layer
- conflict chosen too low-level
Repair:
- use pauses, avoidance, corrected titles, lowered voice, ritual gestures
- shift conflict to responsibility, qualification, procedure, misuse cost
